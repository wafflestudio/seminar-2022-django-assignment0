## 특정 품종의 catfact를 등록하고 / 수정하고 / 삭제하는 Rest API의 URI / METHOD
|      동작      |    METHOD     |               URI                |
|:------------:|:-------------:|:--------------------------------:|
|  등록(CREATE)  |  ```POST```   | /breeds/{breedid}/facts/{factid} |
|  수정(UPDATE)  |   ```PUT```   | /breeds/{breedid}/facts/{factid} |
|  삭제(DELETE)  | ```DELETE```  | /breeds/{breedid}/facts/{factid} |
