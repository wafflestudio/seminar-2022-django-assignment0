새 품종을 등록
```
POST /catfact
request: {
    "breed": ""
}

response: 
- 등록 성공시: HTTP/1.1 201 Created {
    "breed_id" : 1,
    "breed" : ""
}
- 이미 존재하는 품종: HTTP/1.1 409 Conflict {
    "breed" : "",
    "message" : "breed already exist"
}
- 이외: HTTP/1.1 400 Bad Request {
    "message" : "error message"
}
```
